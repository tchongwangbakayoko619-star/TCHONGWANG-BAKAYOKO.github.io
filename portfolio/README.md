# Mon Portfolio

## Description
Ceci est mon portfolio personnel présentant mes compétences, projets et expériences.

## Structure du Projet

```
portfolio/
├── index.html          # Page principale
├── style.css          # Fichier CSS
├── script.js          # Fichier JavaScript
├── assets/            # Dossier pour les médias
│   ├── images/        # Photos, logos
│   └── pdf/           # CV, certificats
├── projects/          # Dossiers pour vos projets
│   ├── projet1/
│   └── projet2/
└── README.md          # Description
```

## Comment utiliser

1. Modifiez `index.html` avec vos informations personnelles
2. Personnalisez `style.css` selon vos préférences
3. Ajoutez vos projets dans le dossier `projects/`
4. Placez vos images dans `assets/images/`
5. Placez votre CV dans `assets/pdf/`

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)

## Installation

Aucune installation requise. Ouvrez simplement `index.html` dans votre navigateur.
